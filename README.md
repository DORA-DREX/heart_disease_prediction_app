# ML Prediction App

A machine learning web app built with Streamlit.

## Features
- Input features through an intuitive interface
- Real-time predictions using trained ML model
- Easy to use

## How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
